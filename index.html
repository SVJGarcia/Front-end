O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação de hipertexto que define a estrutura e o significado do conteúdo de uma página web. É o bloco de construção básico da web. "Hipertexto" refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites. O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador da Web. A marcação HTML inclui "elementos" especiais, como head, title, body, dentre outros.
